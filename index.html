Reletivity Race

I began on Reletivity Race when I started to learn THREE.js. In it, you fly a space ship and are subject to the effects of special relativity. It is currently unfinished.

This project can be found here: Reletivity Race

Kirby Shoot 'em up Fan Game

I really enjoy Kirby games, and I thought kirby would make for a good bullet hell game. This is my attempt at such a game. It is currently unfinished and will likely remain that way for the foreseeable future

This project can be found here: Kirby!

Four Dimensional Shape Viewer

This was my senior project in High School. The goal was to display 2D projections of a selection of 4D shapes. Specifically, I was curious if anaglyph 3D glasses might make the projections easier for the user to understand (ultimately, it didn't help much, but it was cool anyway).

This project can be found here: Four Dimensional Shape Viewer

Chord Diagram

For my senior thesis at OSU, I made a plot showing the correlations between student answer correctness for questions in an introductory physics course. This plot used the D3 library to construct the svg.

The graph can be found here: Chord Diagram
